Release Notes 
-----------------------
OS / CLR Requirements
-----------------------

* PrimoBurner.clr4 - Windows XP SP3 and above / CLR4 (.NET Framework 4.0/4.5)
* PrimoBurner.clr2 - Windows XP SP2 and above / CLR2 (.NET Framework 2.0/3.0/3.5)

------------
Dependencies
------------

PrimoBurner.NET does not depend on the C/C++ runtime libraries.
However it uses PrimoBurner SDK for C++: either PrimoBurner.dll (x86) or PrimoBurner64.dll (x64).

-----------------
Demo Restrictions
-----------------

The read/write speeds are limited as follows:

* CD-R, CD-ROM: 8x
* CD-RW: 4x
* DVD write-once, DVD-ROM: 4x
* DVD rewritable: 2x
* Blu-ray: 1x

These restrictions are removed when a valid license is passed to the Library.

New Features
------------

* DataDisc now supports all UDF revisions up to UDF 2.6.

----------------

* Multi extent UDF files support: The extents collection must contain ALL UDF extents. 
Version 2.5X uses DataFile.FileSize and DataFile.DiscAddress to describe the first UDF extent. 
In version 3.X.X, DataFile.FileSize is the sum of all extents size (i.e. the total file size).

* In version 2.5X, DataDisc.ImageSizeInBlocks reports the loaded image size. 
In version 3.X.X, use the new  DataDisc.LoadedImageSizeInBlocks property.
